# Dogovor
ООО «Токенизация Собеседника» — сервис по предоставлению голосового присутствия на стримах с оплатой по модели токенов.

Репозиторий содержит скрипт для автоматического подсчёта произнесённых слов, их конвертации в токены и расчёта стоимости участия по аналогии с LLM-провайдерами.

Идеально подходит для тех, кто хочет монетизировать разговоры, не заморачиваясь с почасовой оплатой.
